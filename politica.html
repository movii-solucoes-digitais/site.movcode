2. Dados que Coletamos

Para desenvolver seus projetos e manter seus sistemas rodando, coletamos:

  • Dados Cadastrais do Cliente: Nome, e-mail, CPF/CNPJ, telefone e endereço (para fins contratuais e emissão de notas fiscais).
  • Dados dos Seus Usuários: Informações que você ou seus clientes inserem nos sistemas que desenvolvemos (vendas, cadastros, arquivos).
  • Logs Técnicos: Endereço IP, tipo de navegador e timestamps de acesso para garantir a segurança dos sistemas e facilitar o suporte técnico.
  • Comunicação: Histórico de conversas via WhatsApp ou e-mail relacionadas ao suporte e escopo dos projetos.

3. Finalidade do Tratamento

Os dados são utilizados exclusivamente para:

  • Execução do Contrato: Desenvolver o código, realizar o deploy e manter o seu site/sistema funcional.
  • Suporte Técnico: Resolver bugs e realizar manutenções solicitadas.
  • Segurança: Prevenir acessos não autorizados e garantir a integridade do banco de dados.
  • Faturamento: Processar pagamentos recorrentes e emitir documentos fiscais.

4. Compartilhamento (Sub-processadores)

Como agência, utilizamos infraestruturas de ponta para hospedar e processar dados. Seus dados podem passar por:

  • Hospedagem (Cloud): Github pages (armazenamento e processamento).
  • Banco de Dados: MySQL (armazenamento de informações estruturadas).
  • Pagamentos: Asaas para processar suas mensalidades.
  • Comunicação: WhatsApp (via Evolution API ou oficial) e Resend/SendGrid para envios de e-mails do sistema.
  • Inteligência Artificial: Caso seu sistema use IA, utilizamos a OpenAI (com configuração de não-treinamento de modelos).

5. Retenção de Dados

  • Sistemas Ativos: Os dados são mantidos enquanto o contrato de manutenção/hospedagem estiver vigente.
  • Após o Cancelamento: Conforme nossos Termos de Uso, os dados de uso do sistema ficam disponíveis para exportação por 30 dias. Após esse período, realizamos a exclusão definitiva das bases de produção.
  • Obrigação Legal: Dados fiscais e contratuais são preservados por 5 anos para cumprimento de normas da Receita Federal.

6. Seus Direitos (LGPD)

Você, como titular, pode solicitar a qualquer momento:

  • Confirmação e acesso aos dados que tratamos.
  • Correção de informações incompletas ou desatualizadas.
  • Exclusão definitiva de dados (respeitados os prazos de obrigações fiscais).
  • Portabilidade dos dados para outra plataforma.

7. Segurança da Informação

Adotamos práticas rigorosas de desenvolvimento:

  • Uso de criptografia TLS (HTTPS) em todos os acessos.
  • Isolamento de bancos de dados (Row-Level Security) para que um cliente nunca acesse dados de outro.
  • Armazenamento de senhas com hash seguro (bcrypt).
  • Monitoramento constante de logs para identificação de tentativas de invasão.

8. Alterações nesta Política

Sempre que houver uma mudança significativa (como a adição de um novo sub-processador de dados), você será notificado por e-mail ou através de um aviso no painel administrativo do seu sistema.
